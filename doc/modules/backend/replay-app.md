# Replay App（回放服务）

最后更新：2026-06-19

## 概述

`polyedge-replay` 是历史数据回放工具骨架。用于后续重放历史事件序列、验证策略行为或调试数据管道；当前控制台不暴露 `/replay` 页面。

## 设计目标

- 从数据库加载历史事件序列并按时间顺序重放
- 支持自定义回放速度和过滤条件
- 主要用于开发调试和策略验证
