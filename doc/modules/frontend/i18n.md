# i18n（国际化）

最后更新：2026-06-08

## 概述

前端 i18n 系统是自定义实现（非 next-intl），支持 `zh-CN`（默认）和 `en-US` 两种语言。英语是源语言，中文选择性覆盖（缺失键自动回退到英语）。

## 设计目标

- 按命名空间拆分字典，与功能模块对齐
- 运行时可通过 cookie 切换语言
- TypeScript 类型安全：字典结构有 `DeepStringRecord` 类型约束

## 核心 API

### createI18nRuntime(locale)

返回 `{ locale, dictionary, format, enumLabel }`
- `format(key, params?)` — ICU 风格的 `{key}` 插值
- `enumLabel(enumValue)` — 翻译枚举值

### useI18n()

React hook，返回当前 locale 的 dictionary 和 format 函数。通过 `I18nProvider` context 获取。

### Dictionary 类型

`DeepStringRecord<typeof enUS>` — 递归映射类型，保证所有叶节点为 string。

## 使用模式

```typescript
// 组件中
const { format, enumLabel } = useI18n();
format("radar.opportunity.candidate", { status: "candidate" })
enumLabel(signalLifecycleState)

// Server component 中
const dict = getDictionary(locale);
```
